# 📚 StudyHub - Plataforma de Produtividade Educacional

## 🚀 Sobre o Projeto

O **StudyHub** é uma plataforma web moderna desenvolvida para centralizar a organização de materiais de estudo e gerenciamento de trabalhos acadêmicos.

O sistema permite que estudantes organizem disciplinas, armazenem materiais, acompanhem prazos e mantenham produtividade acadêmica em um único ambiente intuitivo.

---

# ✨ Funcionalidades

## 📖 Gerenciamento de Materiais

* Criação de categorias por disciplina
* Upload e armazenamento de arquivos
* Organização por tipo:

  * Resumos
  * Exercícios
  * Provas antigas
  * Anotações
* Sistema de tags para busca rápida
* Favoritar materiais importantes
* Busca inteligente por palavra-chave

---

## 📝 Gerenciamento de Trabalhos

* Cadastro de tarefas e trabalhos
* Definição de prioridade:

  * Baixa
  * Média
  * Alta
* Controle de progresso:

  * Não iniciado
  * Em progresso
  * Concluído
* Controle de prazo de entrega
* Alertas para tarefas próximas
* Anexos e links relacionados

---

## 📊 Dashboard Inteligente

O dashboard inicial apresenta:

* Próximos prazos
* Tarefas urgentes
* Materiais recentes
* Disciplinas cadastradas
* Estatísticas de produtividade
* Barra de pesquisa rápida
